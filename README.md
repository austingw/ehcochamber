# Ehco Chamber

A platform for artists with an emphasis on gatekeeping ;)

Available soon at <a href="https://ehcochamber.com">ehcochamber.com</a>

## What it does

Ehco Chamber allows musicians, collectives, and labels to share their demos, collaborate, and brainstorm within bespoke communities. These "chambers" can be invite only, application based, or open to all. Ehco Chamber also uses Discord for Auth which allows for seamless user flow and community building between the app's upload/feedback features and a Discord server. Futher down the line, fans will be able to support their favorite artists through view-only accounts that grant them access to this exclusive WIP content, insight, and dialogue.

## What we use

Ehco Chamber was built using the <a href="https://create.t3.gg/">T3 stack</a>, <a href="https://ui.shadcn.com/">shadcn/ui</a>, and <a href="https://supabase.com/">Supabase</a>
